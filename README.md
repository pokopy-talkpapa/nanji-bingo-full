# たねまき・とけい（ぜんぶのふん）

対話型算数スクール「たねまき」の授業用シミュレーター。先生が画面共有しながら、**任意の時刻を1分単位で**見せられるツールです。「よーめる」の拡張版。

🔗 **使ってみる**: https://pokopy-talkpapa.github.io/tanemaki-tokei-full/

## 使い方

1. 「なんじ」のプルダウンで時間（0〜12）を選ぶ
2. 「ふん」のプルダウンで分（0〜59）を選ぶ
3. ＋／−ボタンで1分ずつ進めたり戻したりできる
4. 選んだ瞬間に針がアニメーションで動く
5. 下に「○じ ○ふん」が実際の読み方（ふん／ぷん）で大きく表示される

## 表示のオンオフ（トグル）

授業の段階に合わせて、文字盤の要素を出し入れできます。

- **こたえ**：下の「○じ ○ふん」テキスト
- **輪ごと**：真ん中のドーナツ（ヨーメル式の○じエリア）をまるごと
- **○じだけ**：ドーナツの輪・区切りは残し、中の「○じ」文字だけ消す
- **すうじ**：外側の大きな数字（1〜12）。※目盛り・縁線は残るので時計読み練習が成立する

## 特徴

- **1分単位で自由に時刻を作れる**：0時0分〜12時59分まで
- **ヨーメル式文字盤**：内側に「○じ」エリアがあり、短針の先端が指す時刻がひと目で分かる
- **常に時計回り**：戻すときも一周回ってリアル時計と同じ動き
- **長針と短針が連動**：長針1回転＝短針30°ぴったり

## 想定シーン

オンライン授業（スクールタクト併用）で、先生がブラウザでこのページを画面共有 → 子どもは見ながら時計の仕組みを理解。

## シリーズ

「たねまき」授業用デモツール（時計）。「よーめる」（○じ・○じはん）の拡張版。
